# GPTEE.ORG Website

Official website for GPTEE.ORG - A decentralized P2P AI network.

## Tech Stack

- **Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Animations:** Framer Motion
- **Font:** Space Grotesk
- **Deployment:** Vercel

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design Philosophy

- **Terminal Aesthetics:** Developer-focused terminal UI with monospace fonts
- **Color Scheme:** Pure black background with terminal green (#27c93f) accents
- **Font:** Space Grotesk for consistent branding with mobile app
- **Animations:** Smooth transitions and terminal-style effects

## Pages

- **Home:** Landing page with product information and CTAs
- **Nodes:** Real-time node statistics and monitoring dashboard
- **P2P Network:** Infrastructure overview and protocol information
- **Privacy Policy:** Data handling and encryption standards
- **Terms & Conditions:** Usage guidelines and node operator responsibilities

## Future Integrations

- AWS DynamoDB for real-time node statistics
- Live WebSocket connections for node monitoring
- Solana wallet integration for rewards
- Real-time DHT network visualization

## Deployment

Automatically deploys to Vercel on push to main branch.

```bash
# Deploy manually
vercel --prod
```
